Due to changes in how thumbnails etc are handled, Peertube recommends stopping __APP__ and running the `prune-storage` command after upgrading to v8.1.x from v8.0.x, eg:
- `sudo systemctl stop peertube`
- `yunohost app shell peertube`
- `NODE_CONFIG_DIR="__INSTALL_DIR__/config" NODE_ENV=production npm run prune-storage`
- `exit`
- `sudo systemctl start peertube`
